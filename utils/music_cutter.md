# How to use the Music cutter

## 1. Unzip ffmpeg-win64-gpl.7z
- In the directory DiscoverVoice/utils, unzip the ffmpeg-win64-gpl.7z using decompressing tool(like Bandi-zip).

## 2. Run the music_cutter.py
- If you have a Datasets(*.mp3) in the DiscoverVoice/Datasets, you can cut the audio datasets
